# DILS demographic inference outputs

This directory contains the output files from the demographic analyses performed with the **Demographic Inferences with Linked Selection** pipeline (**DILS**) for *Mazzaella laminarioides*. DILS was used to compare alternative divergence scenarios under an Approximate Bayesian Computation framework, including strict isolation, ancient migration, isolation with migration, and secondary contact models.

## Contents

| File                      | Description                                                                   |
| ------------------------- | ----------------------------------------------------------------------------- |
| `sinalelo.tar.gz`         | DILS output archive for the **North versus Center+South** lineage comparison. |
| `centro_sur_alelo.tar.gz` | DILS output archive for the **Center versus South** lineage comparison.       |

## Analysis overview

The DILS analyses were conducted as two independent pairwise comparisons:

1. **North vs Center+South**

   * Tests the first divergence event separating the North lineage from the ancestral Center–South lineage.

2. **Center vs South**

   * Tests the second divergence event separating the Center and South lineages.

For each comparison, DILS performed hierarchical model selection to evaluate whether the data better supported current isolation or ongoing migration, and then compared specific demographic scenarios within the best-supported framework. The analyses also evaluated among-locus heterogeneity in effective population size and migration rates.

## Files included in each archive

Each `.tar.gz` archive contains the DILS output directory generated for one pairwise comparison. Depending on the DILS run, these files may include:

* configuration files used for the analysis;
* model-comparison outputs;
* posterior probability summaries;
* random-forest model-choice summaries;
* parameter estimates for the best-supported model;
* posterior optimization outputs;
* goodness-of-fit and diagnostic files;
* intermediate files generated by the DILS workflow.

## Notes

The final manuscript reports the hierarchical model-choice summaries in Supplementary Table S2 and the optimized posterior parameter estimates for the selected models in Supplementary Tables S3 and S4.

The DILS input consisted of multilocus FASTA files generated from genome-wide 10-kb blocks. Analyses were configured as noncoding regions, and no explicit LD pruning was applied before DILS; therefore, demographic inference was interpreted at the level of multilocus genomic blocks rather than independent SNPs.
